# Anime Shelf

Lightweight local anime watching site. Open `index.html` directly in a browser; no server or external resources are required.

## Features

- Import local video files into a watch list
- Play, search, and filter episodes by status
- Rename the current episode and mark it as watching, done, or planned
- Import a cover image for the current episode
- Save library metadata, covers, and progress in browser local storage

Browsers do not permanently store large local video files for security reasons. After refreshing the page, import the original video again if you want to keep playing it.
